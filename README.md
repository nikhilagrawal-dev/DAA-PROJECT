# Smart Library Search Optimization

A C program that demonstrates and compares library record search performance using Linear Search and Binary Search.

## Features
- Interactive menu-driven terminal app
- Linear Search mode
- Binary Search mode (with sorting step)
- Compare Both mode
- Automatic test cases (best, average, worst)
- Runtime and comparison-count summary dashboard

## File
- `smart_library_search copy.c` - main source code

## Compile
```bash
gcc "smart_library_search copy.c" -o smart_library_search_copy
```

## Run
```bash
./smart_library_search_copy
```

## Notes
- Sorting is required before Binary Search on unsorted data.
- Output is configured to avoid verbose per-step comparison spam.
